사용방법

  • 키보드 입력을 통해 블록을 생성합니다.
  • 생성한 블록은 마우스 드래그를 통해 캔버스 내에서 이동이 가능합니다
  • 블록 끼리 연결하기 위해선 두 블록을 가까이 드래그 하여 붙이고자 하는 블록의 왼쪽 혹은 오른쪽에 경계선의 색이 변하는지 확인합니다.
  • 색이 변하는 걸 확인한 후에 드래그를 풀면 두 블록은 결합 됩니다.
  • 연결 된 블록과 다른 블록 혹은 다른 연결 된 블록끼리 결합도 가능합니다.
  • 블록에 마우스 오른쪽 클릭을 하면 해당 블록을 제어할 수 있는 메뉴가 나옵니다.
  • 해당 메뉴를 나가고 싶다면 캔버스 공간중 메뉴가 아닌 공간 아무 곳이나 클릭하면 메뉴는 사라집니다.
  • execute는 수식을 실행시킵니다.
  • decompse는 수식을 블록 하나로 분해시켜주고, 수식이 아닌 단일 블록의 경우 동작하지 않습니다.
  • copy는 해당 수식 혹은 단일 블록과 동일한 블록을 y+10 좌표에 생성합니다.
  • clean up은 해당 수식 혹은 단일 블록을 삭제합니다.
